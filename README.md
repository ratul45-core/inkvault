# InkVault — Premium Content Platform

A modern, fast, and SEO-optimized content publishing platform built with React, TypeScript, and Vite.

![InkVault Screenshot](https://images.unsplash.com/photo-1499750310107-5fef28a66643?auto=format&fit=crop&q=80&w=1200)

## ✨ Features

- ⚡ **Blazing Fast** - Built with Vite and optimized for Core Web Vitals
- 🌓 **Dark/Light Mode** - Automatic theme switching with system preference
- 📱 **Fully Responsive** - Works perfectly on all devices
- 🔍 **SEO Optimized** - Meta tags, sitemap, robots.txt included
- 🎨 **Beautiful UI** - Tailwind CSS with custom design system
- 📊 **AdSense Ready** - Built-in ad slots for monetization
- 🚀 **Animation** - Smooth Framer Motion animations

## 🛠️ Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Query
- Wouter (routing)
- Lucide Icons

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/inkvault.git

# Navigate to project

# Install dependencies
npm install
